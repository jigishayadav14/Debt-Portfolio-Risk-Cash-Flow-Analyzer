# Debt-Portfolio-Risk-Cash-Flow-Analyzer
Built a Python tool to analyze debt portfolio cash flows and assess risk using a modular design.

Features - 
- Handles debt portfolio data efficiently (data.py)
- Calculates cash flows and schedules (cal.py)
- Evaluates portfolio risk metrics (risk.py)
- Coordinated execution via main.py
- Modular design allows easy reuse and testing of components

Technologies Used - 
- Python 
- Modular programming principles
- Basic financial modeling and risk analysis
